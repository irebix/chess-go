# Phase 2 映射与选择验证摘要

## 自动化覆盖

- `AssetCandidate` 的 code / numericId / name / image 映射；
- 精确偏移、向上 1–4 行和向下 1–3 行扫描；
- 普通英文单词名称不会被误识别为 assetCode；
- 支持 code 前存在额外本地化名称行；
- A 列合并分组多选；一个分组覆盖其右侧全部纵向行段；
- 缺失/非法 assetCode、缺 numericId、缺图片、不支持图片类型；
- 多图片候选按上排参考图、下排项目图展示并默认选择下排；非法或空候选 ID 保持阻断；
- assetCode 重复为 error，numericId 重复为 warning；
- sourceGroup、源表顺序、问题/前缀过滤与四种排序；
- 解析 Manifest 内容；
- 100 项 LRU 缩略图缓存。

分组覆盖检查：两个真实样本中的全部映射项目均被且仅被一个逻辑分组覆盖。连续同名 merge 会合并；组间空白资产段继承前组；首组前空白资产段生成可选的“未命名分组”。

真实结构样例：

- `中国第十五章 / 京菜`：4 个连续同名分段合并为 A51:A70，共 41 项；
- `意大利第二十五至二十八章 / 意大利面系列`：3 个分段合并为 A61:A75，共 29 项；
- `巴西第三十二至三十四章 / 桑巴滋味`：A51:A55 加空白续段 A57:A60，共 49 项；
- 4月、5月、6月和意大利表首部的空标签资产段显示为未命名组，不会丢项。

关键退出样例：

- `M图标月度安排1.xlsx / 第十四章`：30 项，其中 2 个重复 assetCode 项均被 error 阻断；
- `M图标月度安排2.xlsx / 意大利第二十五至二十八章`：229 项，多图片候选默认选择下排项目图；
- `M图标月度安排2.xlsx / 越南第三十九至四十一章`：163 项全部 ready，按 50 项分页为 4 页；
